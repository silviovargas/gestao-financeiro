💾 Fazer backup agora
Baixa uma cópia completa do banco de dados database.db com todos os dados do sistema — usuários, lançamentos financeiros, checklist, metas e permissões. Guarde em local seguro.
Como restaurar: Para restaurar um backup, copie o arquivo .db baixado para /var/www/checklist-app/database.db e reinicie o serviço com sudo systemctl restart ballet-checklist.

Recomendação: Faça backup ao menos uma vez por semana e sempre antes de atualizações do sistema.
📋 Backup automático via terminal
Você também pode fazer backup pelo terminal da VPS a qualquer momento:
cp /var/www/checklist-app/database.db ~/ballet-backup-$(date +%Y%m%d).db
